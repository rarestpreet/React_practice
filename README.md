# React + Vite + Tailwind

This repo contains some projects I used to practice React.js concept.

---

## Prerequisites

Make sure the following are installed on your system:

- Node.js (v16 or above)
- npm (comes with Node.js)

Check installation in system:

```bash
node -v
npm -v
```

## How to run each project (similar steps)

1. Clone the Repositor
```bash
git clone https://github.com/your-username/repo-name.git
cd repo-name/project-name
```

2. Install dependencies
```bash
npm install
```

3. Start Development Server
```bash
npm run dev
```

4. Open in Browser
After running the command, Vite will show something like:<br>
  Local: http://localhost:5173/<br>
Open this URL in browser
